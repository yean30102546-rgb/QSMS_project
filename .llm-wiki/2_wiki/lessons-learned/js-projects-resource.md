## 1. Summary
แหล่งรวมโปรเจกต์ JavaScript (Vanilla JS) 100 โปรเจกต์ ตั้งแต่ระดับเริ่มต้นถึงระดับกลาง เพื่อใช้เป็นแรงบันดาลใจและตัวอย่างการเขียน Logic พื้นฐานโดยไม่ต้องพึ่งพา Framework

## 2. Key Concepts & Use Cases
โปรเจกต์ครอบคลุมหัวใจสำคัญของ JavaScript:
- **DOM Manipulation**: การจัดการโครงสร้างหน้าเว็บ (To-Do List, Calculator)
- **Event Handling**: การดักจับพฤติกรรมผู้ใช้
- **API Fetch & Async/Await**: การดึงข้อมูลจากภายนอก (Random Quote, Weather App)
- **Browser Storage**: การใช้ `localStorage` และ `sessionStorage` เพื่อเก็บสถานะ
- **Date & Time**: การสร้าง Clock, Timer, และ Countdown

## 3. Why Vanilla JS First?
- ช่วยให้เข้าใจการทำงานของ Browser Engine และพื้นฐานของ React/Next.js ได้ลึกซึ้งขึ้น
- ลดความซับซ้อน (No Over-Engineering) สำหรับงานเล็กๆ
- เข้าใจการจัดการ State และ Event ล่วงหน้าก่อนใช้ Framework
