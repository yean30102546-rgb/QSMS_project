## ไฟล์กฎสำคัญ
| ไฟล์ | ตำแหน่ง | บทบาท |

## กฎสรุปย่อ (Quick Reference)
1. **อ่าน `index.md` ก่อนเสมอ** ทุกครั้งที่มีงาน
2. **ห้ามแก้ไข `1_raw/`** — ข้อมูลดิบ Read-Only
3. **ห้ามลบ** ข้อมูลเก่า — ใช้ `[Deprecated]` แทน
4. **ห้ามสร้างไฟล์ซ้ำซ้อน** — รวมและอัปเดตไฟล์เดิม
5. **อัปเดต wiki ทันที** หลังแก้โค้ดหรือแก้ bug สำเร็จ
