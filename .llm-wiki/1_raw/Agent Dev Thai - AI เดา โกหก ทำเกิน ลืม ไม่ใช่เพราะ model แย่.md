[AI เดา โกหก ทำเกิน ลืม ไม่ใช่เพราะ model แย่ แต่เพราะไม่มีใครเขียนกฎห้ามมัน ⸻ กฎพวกนี้เคยโพสต์แยกกันมาหลายรอบ กันโกหก, กัน scope creep, ทำให้มันไม่ลืมข้าม session อยากเก็บไว้ที่เดียว setup ทีเดียวจบ ไม่ต้องไปไล่ก๊อปจากโพสต์เก่าทีละอัน เลยมัดรวมเป็น skill ตัวเดียว setup CLAUDE.md ได้ใน 1 คำสั่ง ข้างในมี 4 ชั้น ⸻ ชั้น 1 · Guardrails: กฎที่หยุด 4 พฤติกรรมพัง 4 กฎ ก๊อปวางลง CLAUDE.md ได้เลย: - NO MAGIC: ไม่รู้ห้ามเดา ไม่รู้ path อยู่ไหนให้ถาม ไม่ใช่แต่งขึ้นมา - VERIFY BEFORE DONE: ห้ามบอก "เสร็จ" ถ้ายังไม่รัน ไม่มีหลักฐาน ห้ามพูด done (กฎสำคัญสุดในไฟล์) - DISSENT: ก่อนทำของใหญ่ ให้เถียงก่อน พังแล้วกระทบแค่ไหน, ถอยกลับได้มั้ย - SCOPE DRIFT: สั่งแก้ bug 1 ตัว แต่ดันไป refactor ทั้ง module ให้มันเตือน - R0/R1/R2: แบ่ง decision ตามการถอยกลับ ถอยไม่ได้ = หยุดถาม, ถอยยาก = ทำแล้วบอก, ถอยง่าย = ทำเลย AI ที่ไม่มีกฎพวกนี้จะทำ 2 แบบ ถามทุกอย่าง (ช้า) หรือทำทุกอย่างเอง (อันตราย) กฎพวกนี้ให้เส้นว่าเมื่อไหร่หยุด เมื่อไหร่วิ่งได้ ⸻ ชั้น 2 · Memory: ทำให้มันไม่ลืมบทเรียน AI ไม่มี memory ข้าม session สอนวันนี้ พรุ่งนี้ลืมหมด พังเรื่องเดิมเป๊ะ วิธีแก้: ไฟล์ MEMORY.md ที่ AI "เขียนเพิ่มเอง" ทุกครั้งที่ทำพลาด แล้ว wire ให้มันโหลดทุก session (ใส่ @MEMORY.md ใน CLAUDE.md) แต่ละ entry มี 3 ช่องเสมอ: เกิดอะไร / ทำไม / ครั้งหน้าทำยังไง ไม่ใช่ "AI พลาดเรื่อง X" (อ่านแล้วทำต่อไม่ได้) ต้องมี root cause กับ correct behavior ที่สั่งตามได้เลย ทุกความผิดเลยกลายเป็นบรรทัดเดียวที่ session หน้าอ่านเอง วงมันแคบลงทุกรอบ ⸻ ชั้น 3 · Spec-driven: /clear แล้วไม่ต้องเล่าใหม่ ชั้น 2 กันลืม บทเรียน แต่ยังมีอีกแบบ ทุก /clear /compact หรือ session crash ลบ สถานะโปรเจค ทิ้ง แล้วต้องนั่งเล่าใหม่ 15-20 นาที "ทำอะไรอยู่ ถึงไหนแล้ว ตกลง architecture ไว้ยังไง" แก้ไม่ใช่ด้วย context window ยาวขึ้น แต่ด้วย spec.md ที่ AI อัพเดทเองหลังทุก task context อยู่ 2 ที่: ใน session (หายตอน /clear, เสื่อมตาม token เหมือน RAM ที่เริ่ม swap) กับใน file (คงที่ ไม่สนว่า session ยาวแค่ไหน) spec.md คือ save point เหมือน save เกมก่อนปิด เปิดมาโหลด save ต่อได้เลย ใส่เป็น rule ใน CLAUDE.md ครั้งเดียว: - เริ่ม session แล้วอ่าน spec.md ก่อนทำอะไร - ทำ task เสร็จ แล้วอัพเดท spec.md (current state, ตัดสินใจอะไรไป, ต่อไปทำอะไร) - ห้ามบอก "เสร็จ" ถ้ายังไม่อัพเดท spec ต้องครบ 2 ฝั่ง เขียนตอนเสร็จ + อ่านตอนเริ่ม spec ที่ไม่มีใครอ่าน = ไดอารี่ spec ที่ไม่มีใครเขียน = เก่าใน 1 วัน ครบทั้งคู่ /clear เลยฟรี เพราะ context ไม่ได้อยู่ใน session มันอยู่ใน file ⸻ ชั้น 4 · Your Brain: ทำให้มันคิดเหมือนคุณ 3 ชั้นแรกใครก็ใช้ได้เหมือนกัน ชั้นนี้ทำให้ AI ทำงานแบบ คุณ สไตล์ตัดสินใจ, น้ำเสียง, งานที่ทำบ่อย ไม่ต้องนั่งเขียนเอง ให้ AI ที่รู้จักคุณอยู่แล้วดึงออกมา เปิดแชทที่คุยบ่อยๆ แล้วสั่ง 3 prompt (อยู่ใน skill เต็มๆ): 1. ดึง work pattern: ตัดสินใจยังไง ชอบ/ไม่ชอบอะไร 2. ดึง voice: โทนเป็นไง คำไหนไม่ใช้เด็ดขาด 3. ดึง skill triggers: พูดคำไหนให้ทำอะไร เอา output วางลง CLAUDE.md จบ มันเลิกเป็น AI ทั่วไป กลายเป็นตัวที่จับวิธีคิดเราได้ ⸻ ก๊อปกฎลงไป + wire memory + วาง spec.md + ดึง pattern ตัวเอง setup ทีเดียว ใช้ได้ทุก session curl -sL https://raw.githubusercontent.com/somnus0x/agt-skill-pack/main/install.sh | bash -s claude-md-setup อยู่ใน agt-skill-pack ฟรีหมด ⸻ CLAUDE.md ที่ว่างเปล่า = AI ที่เริ่มจากศูนย์ทุกเช้า ใส่กฎ ใส่ memory ใส่วิธีคิดคุณลงไป แล้วมันจะไม่เดา ไม่โกหก ไม่ทำเกิน ไม่ลืม ตั้งแต่ session แรก](https://web.facebook.com/permalink.php?story_fbid=pfbid0gD7QT95nH5a7KZtHUdM3R15vCEQKYa8SHriMKzq95QpaX9UzZNHPAfH5173QdKyrl&id=61571959323842)
